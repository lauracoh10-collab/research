# Data Directory Structure

This directory contains all datasets used in the SHAP research project.

## 📁 Directory Structure

```
data/
├── raw/                    # Original, unmodified datasets
│   ├── mimic/             # MIMIC-IV hospital data (not included in repo)
│   ├── censo/             # Brazilian education census (not included in repo)
│   └── .gitkeep
├── processed/              # Cleaned and preprocessed data
│   └── .gitkeep
└── synthetic/              # Generated synthetic datasets
    └── .gitkeep
```

### 3. Synthetic Data

**Location**: `synthetic/`  
**Generated by**: `02_synthetic_data_generation.ipynb`

These datasets are automatically generated with controlled properties:
- Known feature correlations
- Adjustable noise levels
- Ground truth feature importance

**Files**:
- `synthetic_baseline.csv` - Standard synthetic dataset
- `synthetic_high_correlation.csv` - High feature correlation (ρ > 0.8)
- `synthetic_low_noise.csv` - Low noise level
- `synthetic_high_noise.csv` - High noise level

**Generation Example**:
```python
def generate_data(number_samples, number_features, noise):
    """
    Generate synthetic dataset with controlled correlations
    
    Parameters:
    - number_samples: Number of observations
    - number_features: Number of features
    - noise: Noise level for feature generation
    
    Returns:
    - DataFrame with synthetic features
    - List of true feature coefficients
    """
    d = {}
    coefficients = []
    
    # First feature: random normal
    f1 = np.random.randn(number_samples)
    d["f1"] = f1
    
    # Subsequent features: correlated with previous
    for i in range(2, number_features + 1):
        alpha_i = np.random.randn()
        d[f"f{i}"] = alpha_i * d[f"f{i-1}"] + noise
        coefficients.append(alpha_i)
    
    df = pd.DataFrame(d)
    return df, coefficients

# Generate
df_synthetic, true_coefs = generate_data(1000, 5, 0.2)
df_synthetic.to_csv('data/synthetic/synthetic_baseline.csv', index=False)
```

## 🔒 Data Privacy and Security

### IMPORTANT

1. **Never commit raw data to Git**
   - `.gitignore` is configured to exclude `data/raw/`
   - Always verify before pushing

2. **MIMIC-IV Data**
   - Contains protected health information (PHI)
   - Requires PhysioNet Data Use Agreement
   - Only use for approved research purposes

3. **Education Data**
   - May contain sensitive institutional information
   - Check data sharing policies before publishing

4. **Synthetic Data**
   - Safe to share (no real individuals)
   - Can be regenerated from code

## 📝 Data Preprocessing

Processed datasets should be saved in `processed/` with clear naming:

## 📋 Data Loading Utilities

Create a `utils/data_loader.py` for consistent data loading:

```python
import pandas as pd
import os

DATA_DIR = 'data'

def load_mimic_admissions():
    """Load MIMIC-IV admissions data"""
    path = os.path.join(DATA_DIR, 'raw', 'mimic', 'admissions.csv')
    return pd.read_csv(path)

def load_censo_data():
    """Load Brazilian education census"""
    path = os.path.join(DATA_DIR, 'raw', 'censo', 'censo_escolar_2024.csv')
    return pd.read_csv(path)

def load_synthetic(dataset_name='baseline'):
    """Load synthetic dataset"""
    path = os.path.join(DATA_DIR, 'synthetic', f'synthetic_{dataset_name}.csv')
    return pd.read_csv(path)
```

## ❓ FAQ

**Q: Where do I get the MIMIC-IV data?**  
A: Apply for access at https://mimic.mit.edu/. Complete required training.

**Q: Can I share processed MIMIC data?**  
A: No. Even processed data is subject to Data Use Agreement restrictions.

**Q: How large are the datasets?**  
A: 
- MIMIC admissions: ~33,000 rows, ~5MB
- Census data: ~34,000 rows, ~30MB

**Q: Do I need all datasets to run the project?**  
A: No. You can start with synthetic data and gradually add real datasets.
